# 🤖 Binance AI Bot

AI-powered trading bot for Binance Futures, written in Python.

> ⚠️ Educational/research project. Not financial advice. Trading involves risk.

## Features

- Connects to Binance Futures API
- Configurable trading strategies
- AI/ML-assisted signal generation
- Logging and backtesting support (planned)

## Quick Start

```bash
git clone https://github.com/rarl18/binance-ai-bot.git
cd binance-ai-bot
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # add your API keys
python src/bot.py
```
